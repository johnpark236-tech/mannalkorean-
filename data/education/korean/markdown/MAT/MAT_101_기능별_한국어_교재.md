## 핵심개념
- **있음**: 개인적이고 친숙한 상황에서 필요한 대화를 할 수 있음
- **있음**: 정형화된 표현을 사용하거나 두세 번의 말차례를 가진 대화를 할 수 있음
- **있음**: 친숙한 공공장소나 비격식적인 상황에서 필요한 대화를 할 수 있음

- **있음**: 격식적 상황과 비격식적 상황을 구분하여 대화할 수 있음
- **있음**: 복잡한 구조의 대화를 하거나 짧은 내용의 발표를 할 수 있음
- **있음**: 업무 상황이나 공적인 상황에서 격식과 비격식 표현을 구분하여 대화할 수 있음
- **있음**: 간단한 업무 보고나 짧은 분량의 업무 관련 발표를 수행할 수 있음
