| 항목 | 내용 |
|------|------|
| ID | `SEM_033` |
| 교재 | 한국어의미론 |
| 주차·차시 | 3주차 3차시 |
| TOPIK | 미확인 |

## 학습목표
- 검증 대상 단어를 긍정하는 문장을 만듦
- 이어서 그 단어의 의미성분을 부정하는 문장을 만듦<모순문장>
- 그 결과로 앞뒤 두 문장의 의미가 모순이 되면 상정한 의미성분은 타당한 것으로 판
- 검증 대상 단어를 긍정하는 문장을 만듦

## 핵심개념
- **처녀**: - 총각 / 유부남 :
- **의미성분**: 의미성분이 있고 없음을 이분법으로 나타내는 방법
- **아저씨**: [인간], [남자], [어른] / 소년 : [인간], [남자], [아이]
- **아저씨**: [+인간], [+남자], [+어른] / 소년 : [+인간], [+남자], [-어른]
- **소년**: [인간], [남자], [아이] / 소녀 : [인간], [여자], [아이] 소년 : [+인간], [+남자], [-어른] / 소년 : [+인간], [-남자], [-어른]
- **잉여규칙**: 자동적으로 예측 가능한 의미성분 체계
- **잉여성분**: 의미성분들 사이에 자동적으로 예측 가능한 성분
- **아저씨**: [+인간], [+남자], [+어른], [+동물], [+구체물], [-추상물]

## 예시
- ‘부인’ : [인간], [여성], [성숙], [기혼] ž 검증 대상 단어를 긍정하는 문장을 만듦
- ‘부인’ : [인간], [여성], [힘이 셈], [기혼] ž 검증 대상 단어를 긍정하는 문장을 만듦
- ‘죽이다’ : [-게 하다(원인)]x, ([죽다]y) ž 검증 대상 단어를 긍정하는 문장을 만듦
- ‘죽이다’ : [의도한]x ([-게 하다(원인)]x, ([죽다]y)) ž 검증 대상 단어를 긍정하는 문장을 만듦
- 아저씨 : [+인간], [+남자], [+어른], [+동물], [+구체물], [-추상물]
